# 《寻梦》功能演示视频 · 旁白脚本（使用讲解）

- 视频：`寻梦演示.mp4`（约 74 秒，内嵌中文软字幕）｜独立配音：`中文旁白.mp3`｜外挂字幕：`寻梦演示.srt`
- 配音：微软神经网络中文语音 **zh-CN-XiaoxiaoNeural**（自然、有情感，非机械合成），语速 −6%。

1. 欢迎来到寻梦。它能帮你把零散的梦，变成可以重新走进去的画面。我们一起看看怎么用。
2. 在首页点「记录一个梦」，不用注册，也能直接开始。
3. 把你还记得的片段写下来，一句话也行。也可以点语音录入，说出来自动转成文字，之后还能随手修改。
4. 再选一下，此刻这个梦给你的感觉，然后点「编织我的梦」。
5. 稍等十几秒，人工智能就会先织出这个梦的故事；画面也会一张张自动显现、自动播放，不用一直点。
6. 你可以左右切换每一幕，点右上角还能开启朗读和氛围音。看到最后，长按这里，把这个梦轻轻收好。
7. 想留作纪念或者分享，点「分享」，就能把这个梦导出成一张精美的图片。
8. 做过的每一个梦，都会按日历存好，随时回看；在「洞察」里，还能看到你的情绪变化和反复出现的意象。
9. 这就是寻梦——记录、编织，回看你的每一个梦。
